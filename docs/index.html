Frequently Asked Questions

Is 7K Vault really free?

+

Yes! 7K Vault is 100% free and open source under the MIT license. No hidden costs, no premium features, no subscriptions. Everything is free forever.

How secure is the encryption?

+

7K Vault uses AES-256 encryption with PBKDF2 key derivation (100,000 iterations). This is military-grade encryption used by governments and banks. Your password is never stored - only a SHA-256 hash is kept.

What happens if I forget my password?

+

Unfortunately, your files cannot be recovered without the password. This is by design for maximum security. We recommend using a password manager or writing down your password in a safe place.

Does it work offline?

+

Yes! 7K Vault works 100% offline. No internet connection required. No data is sent to any servers. Everything stays on your computer.

Can I use it on multiple computers?

+

Yes! Copy the .secure_vault folder to another computer and use the same password. You can also sync it via external drives or cloud storage (though we recommend encrypted cloud storage).

What file formats are supported?

+

33+ formats: Images (JPG, PNG, GIF, BMP, WebP, HEIC, HEIF, TIFF, ICO), Videos (MP4, AVI, MOV, MKV, WebM, FLV, M4V, WMV, 3GP, MPEG, VOB, OGV), Audio (MP3, WAV, FLAC, M4A, AAC, OGG, WMA, OPUS).

Is there a mobile version?

+

Currently 7K Vault is Windows-only. Mobile versions (Android/iOS) may come in the future. Follow the GitHub repo for updates!

Can I contribute to the project?

+

Absolutely! 7K Vault is open source. Report bugs, suggest features, or submit pull requests on GitHub. All contributions are welcome!
